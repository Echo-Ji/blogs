# 回溯算法的要素

回溯算法跟动态规划算法有些类似，动态规划里强调的是状态，选择和 base case，而回溯法里强调的是路径，选择列表和结束条件。

解决一个回溯问题，其实就是一个决策树的遍历过程，只需考虑以下三个问题：

  • 路径：也就是已经做出的选择
  • 选择列表：即当下可以做的选择
  • 结束条件：到达决策树底层，无法再做选择的条件

# 回溯算法框架

result = []
def backtrace(选择列表, 路径):
    if 满足结束条件:
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrace(选择列表，路径)
