# BFS 解决的问题

BFS 原本是图的一种遍历方式，从一个点开始，向四周开始扩散。一般都是用队列辅助遍历过程，每次将一个节点周围的所有节点加入队列。 因此广泛用于给定起始点（的条件）寻找最短距离。

BFS 相对 DFS 的最主要的区别是：BFS 找到的路径一定是最短的，但代价就是空间复杂度比 DFS 大很多。

下面是用 C++ 写的一个框架，基本思想来源于 labuladong。

        // 当前队列中所有节点向四周扩散
        for(int i = 0; i < sz; ++ i)
        {
            Node cur = q.front();
            q.pop_front();
            
            // 划重点：这里判断是否到达 target
            if(cur is target)
                return step;

            for(Node x: cur.adj())
                if(x not in visited)
                {
                    q.push_back(x);
                    visited.insert(x);
                }
        }
    }
    // 划重点：在这里更新步数
    ++ step;
}
